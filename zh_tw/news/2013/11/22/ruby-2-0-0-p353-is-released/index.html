Ruby 2.0.0-p353 發佈了。

本次發佈修正了浮點數解析的安全性問題。參見：

  • 浮點數解析存在溢出風險 (CVE-2013-4164)

並修復了一些其它的錯誤。

參考 tickets 及 ChangeLog 來了解更多細節。
