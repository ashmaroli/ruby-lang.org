變化

值得一提的變化有：

  • VM (method cache)
  • RGenGC (參見 ko1's RubyKaigi 簡報 及 EuRuKo 簡報)
  • refinements
  • syntax
  • Decimal Literal
  • Frozen String Literal
  • def's return value
  • Bignum
  • 128bit
  • GMP
  • String#scrub
  • Socket.getifaddrs
  • new RubyGems
