Ruby 2.3.4 發布

由 nagachika 發表於 2017-03-30
翻譯： Vincent Lin

Ruby 2.3.4 發布了。

本次發布較前板相比約修正了 80 個錯誤。 參考 commit logs 來進一步了解。

同時本次發布包含了 Symbol#hash 的不確定性修正。 這是 2.3 系列在 2.3.4 版本前存在的功能退化（regression）。 參考 Bug #13376 來進一步了解。

已知問題

（此段落於 2017 年 4 月 11 日加入）

在 Ruby 2.3.4 中發現了 API 不相容，原因是意外刪除了 rb_thread_fd_close 這個 API 函數。 我們會在下一個發布修復這個問題，但如果你遇到了此問題且需要立即解決，請使用這個補丁：
