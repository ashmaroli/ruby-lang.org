Ruby 2.5.9 發佈

由 usa 發表於 2021-04-05
翻譯： twlixin

Ruby 2.5.9 發佈了。

這個版本包含以下的脆弱性修正。 請參照以下的詳細記事。

  • CVE-2020-25613: Potential HTTP Request Smuggling Vulnerability in WEBrick
  • CVE-2021-28965: XML round-trip vulnerability in REXML

另外、配合Ruby的特性，包含了若干変更。 変更的詳細請參照 commit logs。

隨著這個版本的發佈，Ruby 2.5 系列進入了終了階段(EOL)。 這意味著，Ruby 2.5.9 將成為 Ruby 2.5 系列的最後版本。 以後，即使新脆弱性被發現，Ruby 2.5.10 之類的版本也不會被發佈。 我們鼓勵各用戶盡快遷移到更新的版本，例如 3.0、2.7 和 2.6。
