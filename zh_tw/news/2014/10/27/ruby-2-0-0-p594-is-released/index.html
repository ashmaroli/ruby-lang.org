Ruby 2.0.0-p594 發佈了

由 usa 發表於 2014-10-27
翻譯： Juanito Fatas

很高興告訴大家 Ruby 2.0.0-p594 發佈了。

本次發佈包含了 REXML DoS 風險的安全性修正。

  • CVE-2014-8080：XML 擴張的阻斷攻擊

本次發佈也修改了 ext/openssl 的預設設定。危險的 SSL/TLS 選項預設被關閉了。

  • Changed default settings of ext/openssl

同時包含了許多錯誤修正。

參見 tickets 以及 ChangeLog 來了解更多細節。
