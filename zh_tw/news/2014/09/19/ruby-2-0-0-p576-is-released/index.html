Ruby 2.0.0-p576 發佈

由 usa 發表於 2014-09-19
翻譯： Juanito Fatas

很高興宣佈我們發佈了 Ruby 2.0.0-p576，以此慶祝正在日本舉行的 RubyKaigi2014 大會。

本次發佈包含了許多錯誤修正，像是：

  • 修復了許多記憶體洩漏與過量使用的問題；
  • 修正許多與平台相關的問題；
  • 訂正了許多文件錯誤。

參考 Tickets 以及 ChangeLog 來了解更多細節。
