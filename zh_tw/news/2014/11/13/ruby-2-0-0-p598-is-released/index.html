Ruby 2.0.0-p598 發佈了

由 usa 發表於 2014-11-13
翻譯： Juanito Fatas

很高興告訴大家 Ruby 2.0.0-p598 發佈了。

本次發佈包含了 REXML DoS 風險的安全性修正。 和上一版已經修正的風險類似，但本次處理了實體擴張另外的情況。請參考下面的文章來了解更多細節。

  • CVE-2014-8090：另一個 XML 擴張導致的阻斷攻擊

同時包含了一些錯誤修正。

參考 tickets 和 ChangeLog 來了解更多細節。
