Ruby 2.3.6 Rilis

Ditulis oleh usa tanggal 2017-12-14
Diterjemahkan oleh meisyal

Ruby 2.3.6 telah dirilis.

Rilis ini mencakup perbaikan 10 bug dari rilis sebelumnya, dan juga beberapa perbaikan keamanan. Mohon cek topik-topik di bawah ini untuk lebih detail.

  • CVE-2017-17405: Command injection vulnerability in Net::FTP
  • Unsafe Object Deserialization Vulnerability in RubyGems

Lihat ChangeLog untuk detail.
