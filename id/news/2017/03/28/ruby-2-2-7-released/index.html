Ruby 2.2.7 Rilis

Ditulis oleh usa tanggal 2017-03-28
Diterjemahkan oleh meisyal

Ruby 2.2.7 telah dirilis.

Rilis ini mencakup sekitar 70 perbaikan bug setelah rilis sebelumnya. Lihat ChangeLog untuk detail.

Setelah rilis ini, kami akan mengakhiri fase perawatan yang biasa dilakukan pada Ruby 2.2, dan memulai fase perawatan keamanannya. Ini berarti setelah versi 2.2.7 rilis kami tidak akan pernah backport perbaikan bug apapun untuk versi 2.2 kecuali perbaikan keamanan. Masa perawatan keamanan dijadwalkan selama setahun. Setelah masa tersebut berakhir, dukungan resmi dari Ruby 2.2 akan berakhir. Sehingga, kami merekomendasikan Anda untuk mulai merencanakan upgrade ke Ruby 2.4 atau 2.3.
