Ruby 1.9.3-p125 dirilis.

Rilis ini termasuk perbaikan keamanan dari ekstensi OpenSSL Ruby OpenSSL. Dan banyak bug yang diperbaiki pada rilis ini.

== Perbaikan

  • Perbaikan untuk modul OpenSSL Ruby: Izinkan "0/n splitting" untuk pencegahan serangan TLS BEAST
  • Diperbaiki: Dukungan LLVM/clang [Bug #5076]
  • Diperbaiki: Dukungan GCC 4.7 [Bug #5851]
  • Perbaikan bug lain

Lihat tiket dan ChangeLog untuk rinciannya.
