Ruby 2.2.1 Rilis

Ditulis oleh hsbt tanggal 2015-03-03
Diterjemahkan oleh meisyal

Kami dengan senang hati mengumumkan rilis dari Ruby 2.2.1. Ini adalah versi rilis TEENY pertama dari rangkaian versi 2.2 yang stabil.

Rilis ini mencakup perbaikan untuk ffi build failure dan memory leak pada Symbol GC (Lihat Bug #10686).

Lihat ChangeLog untuk detail.
