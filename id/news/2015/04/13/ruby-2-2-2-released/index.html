Ruby 2.2.2 Rilis

Ditulis oleh nagachika tanggal 2015-04-13
Diterjemahkan oleh meisyal

Kami dengan senang hati mengumumkan rilis dari Ruby 2.2.2. Ini adalah sebuah versi TEENY dari rangkaian stable 2.2.

Rilis ini mencakup perbaikan keamanan untuk OpenSSL extension's hostname verification vulnerability.

  • CVE-2015-1855: Ruby OpenSSL Hostname Verification

Ada juga beberapa bugfixes. Lihat ChangeLog untuk detail.
