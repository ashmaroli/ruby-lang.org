Ruby 2.0.0-p353 dirilis

Ditulis oleh nagachika tanggal 2013-11-22
Diterjemahkan oleh gozali

Sekarang Ruby Ruby 2.0.0-p353 telah dirilis.

Rilis ini termasuk sebuah perbaikan keamanan tentang floating point parsing.

  • Heap Overflow pada Floating Point Parsing (CVE-2013-4164)

Dan beberapa bugfix juga dimasukan. Lihat tickets dan ChangeLog untuk rincian.
