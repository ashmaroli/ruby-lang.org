Ruby 2.1.0-preview2 telah dirilis

Ditulis oleh nurse tanggal 2013-11-22
Diterjemahkan oleh gozali

Kami sangat senang untuk mengumumkan rilis dari Ruby 2.1.0-preview2. Silakan uji fitur baru di Ruby 2.1 sebelum rilis akhir!

Perubahan penting dari preview 1

  • perbaikan Heap Overflow dalam Floating Point Parsing (CVE-2013-4164)
  • "literal".freeze sekarang dioptimalkan #9042
  • f suffix dari String Literal dihapus #9042
  • perbaikan isu memory consuming pada RGenGC (r43532 dan r43755)
  • penambahan Exception#cause #8257
  • pemutakhiran pustaka seperti json, nkf, rake, RubyGems, dan RDoc.

Perubahan dari 2.0

Perubahan penting adalah:

  • VM (method cache)
  • RGenGC (Lihat presentasi ko1's di RubyKaigi dan presentasi RubyConf 2013)
  • perbaikan #8481 #8571
  • perubahan syntax
    • Rational/Complex Literal #8430
    • def's return value #3753
  • Bignum
    • menggunakan 128bit integers #8509
    • menggunakan GMP #8796
  • String#scrub #8414
  • Socket.getifaddrs #8368
  • RDoc 4.1.0.preview.2 dan RubyGems 2.2.0.preview.2

Lihat rincian perubahan: NEWS di repository Ruby (WIP).
