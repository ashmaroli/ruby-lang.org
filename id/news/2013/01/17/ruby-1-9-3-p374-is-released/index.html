Ruby 1.9.3-p374 Dirilis

Ditulis oleh usa tanggal 2013-01-17
Diterjemahkan oleh catcyborg

Now Ruby 1.9.3-p374 dirilis.

Rilis ini mencakup banyak perbaikan bug. Terutama,

  • Memperbaiki masalah SEGV acak (sering dilaporkan dengan Rails)
  • Dukungan Windows 8 (mungkin, semoga)
  • perbaikan bug lain.

Lihat tiket dan ChangeLog untuk rinciannya.

Perbaikan keamanan baru tidak termasuk dalam rilis ini, tetapi saya sarankan pengguna versi sebelumnya (p362) untuk meng-update.
