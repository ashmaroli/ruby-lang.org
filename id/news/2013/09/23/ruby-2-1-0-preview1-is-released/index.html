Perubahan

Perubahan penting pada versi ini adalah:

  • VM (metode cache)
  • RGenGC (Lihat presentasi ko1 di RubyKaigi dan presentasi EuRuKo)
  • refinements
  • sintaks
  • Decimal Literal
  • Frozen String Literal
  • def's return value
  • Bignum
  • 128bit
  • GMP
  • String#scrub
  • Socket.getifaddrs
  • new RubyGems

Lihat rincian lebih detail dari perubahan: NEWS di Ruby repository (WIP).
