Ruby 1.9.3-p429 Diluncurkan

Ditulis oleh usa tanggal 2013-05-14
Diterjemahkan oleh catcyborg

Sekarang Ruby 1.9.3-p429 diluncurkan. Kami telah merilis p426 beberapa jam sebelumnya, tetapi versi itu memiliki masalah build pada beberapa platform. Harap gunakan p429.

Rilis ini berisi perbaikan pada security dari extension Ruby DL / Fiddle yang dibundel.

  • Taint Object mem-bypass DL dan Fiddle pada Ruby (CVE-2013-2065)

Beberapa perbaikan bug juga termasuk dalam rilis ini.

Lihat tiket and changelog untuk rinciannya.
