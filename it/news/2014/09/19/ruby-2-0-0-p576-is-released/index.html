Rilasciato Ruby 2.0.0-p576

Inserito da usa il 2014-09-19
Tradotto da alepore

Siamo lieti di annunciare il rilascio di Ruby 2.0.0-p576 che celebra il RubyKaigi2014 che si sta tenendo ora in Giappone.

Questa release include molti bugfix come:

  • numerosi fix di memory leak e uso eccessivo di memoria,
  • numerosi fix di problemi platform-specific (specialmente nel processo di build),
  • numerosi fix alla documentazione.

Vedere i ticket e il ChangeLog per i dettagli.
