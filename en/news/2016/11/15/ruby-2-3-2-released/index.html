Ruby 2.3.2 Released

Posted by nagachika on 15 Nov 2016

Ruby 2.3.2 has been released.

This is a TEENY version release of the stable 2.3 series.

This release contains an update to RubyGems 2.5.2 and updates of the included SSL certificates.

There are many bugfixes, too. See the ChangeLog for details.
