Ruby 2.3.3 Released

Posted by nagachika on 21 Nov 2016

Ruby 2.3.3 has been released.

This release contains a bug fix concerning Refinements and Module#prepend. The combined use of Module#refine and Module#prepend to the same class could cause an unexpected NoMethodError. This is a regression on Ruby 2.3.2 released last week. See Bug #12920 for details.

There are some other bugfixes, too. See the ChangeLog for details.
