Ruby 2.0.0-p648 Released

Posted by usa on 16 Dec 2015

Ruby 2.0.0-p648 has been released.

This release includes a security fix for the Fiddle and DL extensions. Please view the topic below for more details.

  • CVE-2015-7551: Unsafe tainted string usage in Fiddle and DL

Ruby 2.0.0 is now under the state of the security maintenance phase, until Feb. 24th, 2016. After that date, maintenance of Ruby 2.0.0 will be ended. We recommend you start planning migration to newer versions of Ruby, such as 2.1, 2.2 or 2.3 (scheduled to release within a few weeks).
