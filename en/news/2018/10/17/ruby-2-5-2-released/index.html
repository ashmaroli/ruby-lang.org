Ruby 2.5.2 Released

Posted by nagachika on 17 Oct 2018

Ruby 2.5.2 has been released.

This release includes some bug fixes and some security fixes.

  • CVE-2018-16396: Tainted flags are not propagated in Array#pack and String#unpack with some directives
  • CVE-2018-16395: OpenSSL::X509::Name equality check does not work correctly

There are also some bug fixes. See the commit logs for more details.
