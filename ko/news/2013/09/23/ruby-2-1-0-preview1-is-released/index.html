변경점

주목할 만한 변경점들은 아래와 같습니다:

  • VM (메서드 캐시)
  • RGenGC (ko1의 RubyKaigi presentation과 EuRuKo presentation을 참조하십시오)
  • refinements
  • syntax
  • Decimal Literal
  • Frozen String Literal
  • def의 return value
  • Bignum
  • 128bit
  • GMP
  • String#scrub
  • Socket.getifaddrs
  • 새로운 RubyGems

보다 상세한 변경점들은 루비 저장소의 뉴스(작업 중)를 참조하십시오.

ko1(Koichi Sasada)이 toruby에서 발표한 내용입니다: Summary of Ruby 2.1

Konstantin Haase(@konstantinhaase)가 멋진 요약을 자신의 블로그 포스트로 작성하였습니다: What's new in Ruby 2.1?.
