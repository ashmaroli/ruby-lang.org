루비 2.1.0-preview2 릴리스

작성자: nurse (2013-11-22)
번역자: SeungKyun Nam

루비 2.1.0-preview2의 릴리스를 발표하게 되어 기쁩니다. 최종 릴리스 전에 루비 2.1의 새로운 기능들을 테스트해 주십시오.

preview 1 이후의 주목할 만한 변화

  • 부동소수점 파싱할 때 힙 오버플로 발생 (CVE-2013-4164) 수정함
  • "literal".freeze 최적화 #9042
  • 문자열의 f 접미사 제거 #9042
  • RGenGC에서 메모리 소모 이슈 (r43532 그리고 r43755) 수정
  • Exception#cause #8257 추가
  • json, nkf, rake, RubyGems, 그리고 RDoc 등의 라이브러리 업데이트

2.0 이후의 변화들

주목할만한 변화는 아래와 같습니다:

  • VM (메서드 캐시)
  • RGenGC (ko1의 RubyKaigi presentation과 RubyConf 2013 presentation을 참조하십시오)
  • refinements #8481 #8571
  • 문법 변화
    • Rational/Complex Literal #8430
    • def의 return value #3753
  • Bignum
    • 128비트 정수 사용 #8509
    • GMP 사용 #8796
  • String#scrub #8414
  • Socket.getifaddrs #8368
  • RDoc 4.1.0.preview.2와 RubyGems 2.2.0.preview.2

보다 상세한 변경점들은 루비 저장소의 뉴스(작업 중)를 참조하십시오.

ko1이 toruby에서 루비 2.1의 새로운 기능들에 대해 언급했습니다: All about Ruby 2.1

Konstantin Haase(@konstantinhaase)가 자신의 블로그 포스트에 멋진 요약본을 작성하였습니다: What's new in Ruby 2.1?.
