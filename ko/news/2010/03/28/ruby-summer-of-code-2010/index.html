Ruby Summer of Code 2010

작성자: Moru (2010-03-28)

'Ruby Summer of Code 2010'(이하 RubySOC)이 개최됩니다. 기존까지 Google Summer of Code의 일부로서 개최가 되었으나 올해는 독립적으로 개최될 예정입니다. RubySOC는 Ruby Central, Engine Yard, the Rails team이 공동으로 기획에 참여하였으며, RubySOC에 관련한 자세한 사항은 rubysoc.org와 twitter@rubysoc에서 확인할 수 있습니다.

기존 Google Summer of Code의 구성과 동일하게 진행될 이번 RubySOC는

  • 채택작에 개발지원금 $5,000를 지원
  • 루비 guru의 멘토 제도
  • 평가기준: 실용성, 루비 커뮤니티에 대한 수혜, 오픈소스에 대한 공헌 내력

와 같은 내용을 주요 골격으로 삼고 있습니다.

응모는 오는 4월 4일부터 시작됩니다. 관심있는 분은 rubysoc.org에서 자세한 정보를 살펴보시기 바랍니다.

이외에도 루비와 Jruby, 레일즈에서 제안한 아이디어를 한 번 살펴보시기 바랍니다.

최근 소식

  • Ruby 3.2.0 릴리스
  • Ruby 3.2.0 RC 1 릴리스
  • Ruby 3.1.3 릴리스
  • Ruby 3.0.5 릴리스
  • Ruby 2.7.7 릴리스
