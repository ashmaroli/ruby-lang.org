Ruby 2.5.9 릴리스

작성자: usa (2021-04-05)
번역자: yous

Ruby 2.5.9가 릴리스되었습니다.

이 릴리스는 보안 수정을 포함합니다. 자세한 사항은 아래 글을 확인해보세요.

  • CVE-2020-25613: WEBrick의 잠재적인 HTTP 요청 스머글링 취약점
  • CVE-2021-28965: REXML의 XML 왕복 변환(round-trip) 취약점

자세한 사항은 커밋 로그를 확인해주세요.

이 릴리스 이후로 Ruby 2.5의 지원이 종료됩니다. 즉, 이는 Ruby 2.5 버전대의 마지막 릴리스입니다. 보안 취약점이 발견되어도 Ruby 2.5.10이 릴리스되지 않습니다. 모든 Ruby 2.5 사용자는 Ruby 3.0, 2.7, 2.6으로 즉시 업그레이드하기 바랍니다.
