sourceforge에 올라온 Windows용 루비 인스톨러

작성자: NaHi (2002-11-25)

이제 Windows용 루비 인스톨러를 http://rubyinstaller.sourceforge.net/에서 구할 수 있습니다.

사이트에서 발췌했습니다.

이것은 루비 프로그램 자체와 수십 가지의 인기있는 확장기능,
패키지, 구문강조 에디터, 실행 환경, "Programming Ruby:
The Pragmatic Programmer's Guide" 책 전부의 Windows 도움말
파일을 포함하는 "원 클릭" 자체 설치 프로그램입니다.

이것은 Win32 박스에서 루비를 사용하는 분께는 머스트 해브 아이템입니다. Win32 바이너리와 수십여 개의 인기있는 익스텐션을 보시려면 usa의 바이너리와 eban의 win32 바이너리도 체크하세요.

최근 소식

  • Ruby 3.2.0 릴리스
  • Ruby 3.2.0 RC 1 릴리스
  • Ruby 3.1.3 릴리스
  • Ruby 3.0.5 릴리스
  • Ruby 2.7.7 릴리스
