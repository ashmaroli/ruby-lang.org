Google Summer of Code 2015

작성자: Federico Builes (2015-03-06)
번역자: yous

루비는 Google Summer of Code 2015에 최상위 조직으로 참가할 것입니다. 우리는 루비, JRuby, 셀룰로이드와 루비와 관련된 다른 프로젝트의 우산 역할을 할 것입니다. 학생 신청은 3월 16일부터 3월 27일까지 받을 것입니다(일정표는 여기 있습니다).

학생이나 멘토로 참여하고 싶다면 RubyGSoC 메일링 리스트에 들어오시기 바랍니다. RubyGSoC 위키에서 프로젝트에 관한 아이디어를 볼 수 있습니다.

루비 온 레일즈와 SciRuby 역시 최상위 조직으로서 올해 프로그램에 참여합니다. 이들 프로젝트에 관해 좋은 아이디어가 있다면 각 프로젝트의 발표문을 읽어보시기 바랍니다.

최근 소식

  • Ruby 3.2.0 릴리스
  • Ruby 3.2.0 RC 1 릴리스
  • Ruby 3.1.3 릴리스
  • Ruby 3.0.5 릴리스
  • Ruby 2.7.7 릴리스
