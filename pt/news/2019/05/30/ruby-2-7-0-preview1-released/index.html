Lançado Ruby 2.7.0-preview1

Escrito por naruse em 30/05/2019
Traduzido por jcserracampos

Temos o prazer de anunciar o lançamento da versão de Ruby 2.7.0-preview1.

Uma versão de pré-visualização (preview) é lançada para coletar opiniões para a versão final planejada para ser lançada em dezembro. Ela introduz várias novas funcionalidades e melhorias de performance, mais notavelmente:

  • Compactação no coletor de lixo
  • Pattern Matching
  • Melhoria do REPL

Compactação no coletor de lixo (Compaction GC)

Esta versão introduz compactação na coleta de lixo (Compaction GC) que pode desfragmentar um espaço de memória fragmentado.

Alguns programas multithread Ruby podem causar fragmentação de memória, conduzindo a alto uso de memória e velocidade degradada.

O método GC.compact é introduzido para compactar a heap. Esta funcionalidade compacta objetos ativos na heap para que menos páginas possam ser usadas e a heap possa ser mais CoW friendly. #15626

Pattern Matching [Experimental]

Pattern matching, funcionalidade amplamenta utilizada em linguagens para programação funcional, é introduzida como uma funcionalidade experimental. #14912 Ela pode examinar um dado objeto e definir seu valor se um padrão for.

json ='{
  "nombre": "Alice",
  "edad": 30,
  "hijos": [
    {
      "nombre": "Bob",
      "edad": 2
    }
  ]
}'
case JSON.parse('{...}', symbolize_names: true)
in {name: "Alice", children: [{name: "Bob", age: age}]}
  p age
  ...
end

Para mais detalhes, por favor, veja Pattern matching - New feature in Ruby 2.7.

Aprimoramento de REPL

irb, ambiente interativo empacotado (REPL; Read-Eval-Print-Loop), agora suporta edição em múltiplas linhas. É fomentado por reline, implementação pura em Ruby compatível de readline. Também prove integração com rdoc. Em irb você pode exibir a referência para uma dada classe, módulo ou método. #14683, #14787, #14918 Além de, linhas de código mostradas em binding.irb e resultados de inspeções para objetos de classes principais agora são colorizados.

Outras novas funcionalidades notáveis

  • Um operador de referência de método, .:, é introduzido como uma característica experimental. #12125, #13581

  • Parâmetro numerado como o parâmetro de bloco padrão é introduzida como uma característica experimental. #4475

  • Um intervalo sem começo é introduzido experimentalmente. Pode não ser tão útil como um intervalo sem fim, mas seria bom para o propósito do DSL. #14799

    ary[..3]  # indêntico a ary[0..3]
rel.where(sales: ..100)

  • Enumerable#tally é adicionado. Ele conta a ocorrência de cada elemento.

    ["a", "b", "c", "b"].tally
#=> {"a"=>1, "b"=>2, "c"=>1}

Melhorias de desempenho

  • JIT [Experimental]

    • Código em JIT é recompilado para código menos otimizado quando uma suposição de otimização é invalidada.

    • Inlining de método é executado quando um método é considerado puro. Essa otimização ainda é experimental e muitos métodos NÃO são considerados puros ainda

    • Valor padrão de --jit-min-calls é alterado de 5 para 10,000

    • Valor padrão --jit-max-cache é alterado de 1,000 para 100

Outras mudanças notáveis desde 2.6

  • Proc.new e proc sem bloco em um método invocado com um bloco é alertado agora.

  • lambda sem bloco em um método invocado com um bloco produz um erro.

  • Atualiza versão do Unicode e Emoji de 11.0.0 para 12.0.0. [Feature #15321]

  • Atualiza versão do Unicode para 12.1.0, adicionando suporte para U+32FF SQUARE ERA NAME REIWA. [Feature #15195]

  • Date.jisx0301, Date#jisx0301 e Date.parse provisoriamente suportam a nova era Japonesa como uma extensão informal, até o novo JIS X 0301 seja emitido. [Feature #15742]

  • Requere compiladores para suportar C99 [Misc #15347]

    • Detalhes do nosso dialeto: https://bugs.ruby-lang.org/projects/ruby-master/wiki/C99

Veja NEWS ou commit logs para mais detalhes.

Com essas mudanças, 1727 arquivos modificados, 76022 inserções(+), 60286 remoções(-) desde Ruby 2.6.0!

Aproveite programando com Ruby 2.7!

O que é Ruby?

Ruby foi primeiramente desenvolvido por Matz (Yukihiro Matsumoto) em 1993 e agora é desenvolvido como Open Source. Ele é executado em múltiplas plataformas e é usado em todo o mundo, especialmente para desenvolvimento web.
