Lançado Ruby 2.6.3

Escrito por naruse em 17/04/2019
Traduzido por jcserracampos

Ruby 2.6.3 foi lançado.

Esta versão adiciona suporte para a nova era japonesa "令和" (Reiwa). Ela atualiza a versão do Unicode para 12.1 beta (#15195) e atualiza a biblioteca de data (#15742).

Esta versão tanbém inclui algumas correções de bug. Veja os commit logs para detalhes.
