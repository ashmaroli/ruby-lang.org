Ruby 2.0.0-p451 is released

Escrito por nagachika em 24/02/2014
Traduzido por diogoandre

Hoje, 24 de Fevereiro, é o 21º aniversário do Ruby, e com felicidade anunciamos uma nova versão patch, Ruby 2.0.0-p451.

Esta versão inclui muitas correções de bugs. Veja tickets e ChangeLog para detalhes.
