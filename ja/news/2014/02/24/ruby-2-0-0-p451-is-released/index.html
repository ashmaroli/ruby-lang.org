Ruby 2.0.0-p451 リリース

Posted by nagachika on 24 Feb 2014

今日は Ruby の 21 回目の誕生日です。 これを記念して、Ruby 2.0.0-p451 がリリースされました。

今回のリリースには、多数のバグ修正が含まれています。 詳しくは、対応するチケットおよび ChangeLog を確認して下さい。
