Ruby 1.9.3-p125 リリース

Posted by NARUSE, Yui on 16 Feb 2012

Ruby 1.9.3-p125 がリリースされました。

このリリースには Ruby OpenSSL 拡張ライブラリへのセキュリティフィックス が含まれています。 また、たくさんのバグが修正されています。

== Fixes

  • TLSの脆弱性に対する攻撃（TLS BEAST攻撃）対策のためのOpenSSL機 能、"0/n splitting"を利用できるようにしました。
  • 修正: LLVM/clang サポート [Bug #5076]
  • 修正: GCC 4.7 サポート [Bug #5851]
  • その他のバグ修正

See tickets and ChangeLog for details.
