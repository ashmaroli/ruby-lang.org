このウェブサイトについて

このウェブサイトは Jekyll を用いて Ruby で生成されています。 また、そのソースコードは GitHub で管理されています。

サイトのビジュアルデザインは Jason Zimdars によるものです。 Ruby Visual Identity Team による初期のデザインが元になっています。

Ruby のロゴ はまつもとゆきひろの著作物です。 (Copyright © 2006, Yukihiro Matsumoto)

問題の報告

問題の報告には、issue trackerを使うかウェブマスターに連絡してください。 (英語で記述してください)

協力するには

このウェブサイトは Ruby コミュニティーのメンバーによって運営されています。

もしあなたが協力したい場合、 contribution instructions を読んでから issue や pull request を送ってください！
