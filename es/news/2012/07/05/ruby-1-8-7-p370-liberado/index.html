Ruby 1.8.7-p370 liberado.

Publicado por Manuel Ramos Caro el 2012-07-05

Con esta versión se inicia la fase final de vida de la versión 1.8.7 de Ruby. Por lo que si todavia usas esta rama te deberas plantear empezar a utilizar Ruby 1.9.

A continuación os detallo las fases de planificación del fin de ciclo de vida de Ruby 1.8.7:

  • Se continuara facilitando mantenimiento normal de Ruby 1.8.7 hasta Junio de 2012. Se corregiran errores y se asegurara la compatibilidad.
  • Despues de esto se pararan las correcciones. Se continuaran facilitando correcciones de seguridad hasta Junio de 2013.
  • Una vez finalizo este periodo se dara por cerrado el ciclo de vida de la rama finalizando el soporte de esta.

Con esta versión de Ruby se cumple el primer punto de la citada hoja de ruta.
