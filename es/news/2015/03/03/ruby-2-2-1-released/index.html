Liberada la versión 2.2.1 de Ruby

Publicado por hsbt el 2015-03-03
Traducción de David Padilla

Estamos emocionados de anunciar la entrega de Ruby 2.2.1. Este es la primer versión TEENY de la serie estable de 2.2.

Esta entrega contiene la corrección para la falla de compilación de ffi y una fuga de memoria en el colector de objetos tipo Symbol (Ver Bug #10686).

También puedes leer el ChangeLog para obtener más detalles.
