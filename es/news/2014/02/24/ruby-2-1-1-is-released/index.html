Liberado Ruby 2.1.1

Publicado por naruse el 2014-02-24
Traducción de David Padilla

Hoy, Febrero 24 es el 21 aniversario de Ruby así que estamos muy contentos de anunciar una nueva entrega de Ruby 2.1, Ruby 2.1.1.

Ruby 2.1 tiene muchas mejoras incluyendo algunas de velocidad sin incompatibilidades mayores. Puedes usar esta version en Rails y otras aplicaciones y obtener una experiencia mas agradable.

Esta entrega incluye varias correcciones. Ver los tickets y el ChangeLog para conocer los detalles.

Después de anunciar el cambio en la política de versiones de Ruby 2.1, esta versión será simplemente llamada "2.1.1"esta versión será simplemente llamada "2.1.1".

Actualización: Encontramos una regresión en Hash#reject. Para mas detalles, ver: Regresión en Hash#reject en Ruby 2.1.1.
