Liberada la versión de Ruby 2.0.0-p598

Publicado por usa el 2014-11-13
Traducción de David Padilla

Hemos liberado la versión de Ruby 2.0.0-p598.

Esta entrega contiene una corrección de seguridad de una vulnerabilidad por negación de servicio en REXML. Es muy similar a la vulnerabilidad corregida en la entrega anterior pero en este caso manejamos un caso distinto de la expansión de entidades. Por favor continua leyendo para obtener más detalles.

  • CVE-2014-8090: Otro ataque por negación de servicio en la expansión de XML

También se incluyen otras correciones. Puedes ver los tickets y el ChangeLog para tener una lista completa de los cambios incluidos en la versión.
