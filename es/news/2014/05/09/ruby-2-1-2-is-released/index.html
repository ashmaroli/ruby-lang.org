Liberada la versión de Ruby 2.1.2

Publicado por nagachika el 2014-05-09
Traducción de David Padilla

Nos sentimos complacidos de anunciar la entrega de Ruby 2.1.2.

Esta entrega contiene correcciones de una regresión en Hash#reject en Ruby 2.1.1, soporte para compilar con Readline-6.3 (ver Bug #9578), una versión actualizada de libyaml con psych y algunas otras correcciones.

Ver los tickets y el ChangeLog para mas detalles.
