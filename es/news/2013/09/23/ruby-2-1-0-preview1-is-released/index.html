Liberado Ruby 2.1.0-preview1

Publicado por nurse el 2013-09-23
Traducción de David Padilla

Cambios

Los cambios que sobresalen en esta versión son:

  • VM (caché de métodos)
  • RGenGC
  • refinements
  • sintaxis
  • Literal decimal
  • Literales de cadena congelados
  • el retorno de valor de def
  • Bignum
  • 128bit
  • GMP
  • String#scrub
  • Socket.getifaddrs
  • Nueva RubyGems
