Ruby 2.0.0-p451 今日发布

由 nagachika 发表于 2014-02-24
翻译： P.S.V.R

今天，2月24日，是Ruby的21周岁生日， 因此，我们很高兴地宣布发布一个新的补丁版本和Ruby 2.0.0，Ruby 2.0.0-p451。

这次发布包含了许多缺陷修复。 见 tickets 与 ChangeLog 以了解更多细节。
