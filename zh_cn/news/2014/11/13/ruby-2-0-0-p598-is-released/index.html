Ruby 2.0.0-p598 发布

由 usa 发表于 2014-11-13
翻译： cassiuschen

我们很高兴地宣布 Ruby 2.0.0-p598 发布了。

本次发布包括了一个针对 REXML 拒绝服务攻击（DoS）漏洞的安全修复。 下述日志中包含相似内容： 预发布中的漏洞修复, 但这次的发布中解决了 XML Entity Expansion的问题。 请看下述文章以了解详情。

  • CVE-2014-8090：另一个 XML 洪水攻击

此外，本次发布还包含以下修复： 请见 tickets 和 ChangeLog 以了解详情。
