谷歌编程之夏 2015

由 Federico Builes 发表于 2015-03-06
翻译： Kevin Cheng

Ruby 将以顶级组织身份参与Google Summer of Code 2015。我们将为包括 Ruby、JRuby、Celluloid 和其他这样的 Ruby 相关项目提供保护伞。学生项目提交日期为 3 月 16 日至 3 月 27 日（这是时间表）。

请有兴趣参加本次活动的学生或老师加入邮件列表。在RubyGSoC wiki有一份相关项目的列表。

Ruby on Rails 和 SciRuby 同样以顶级组织的身份参加本次活动。如果有更适合这些项目的想法，请查看相应的公告 Ruby on Rails、SciRuby。
