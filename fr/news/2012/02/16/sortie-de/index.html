Sortie de Ruby 1.9.3-p125

Posté par Bruno Michel le 2012-02-16

La version 1.9.3-p125 de Ruby est sortie. Elle inclut un correctif de sécurité pour l'extension OpenSSL de Ruby et corrige d'autres bugs.

Correctifs

  • Correctif pour le module Ruby OpenSSL : Correction d'une faille de sécurité dans le module OpenSSL: "0/n splitting" comme mesure préventive contre l'attaque TLS BEAST
  • Corrigé : prise en charge de LLVM/clang [Bug #5076]
  • Corrigé : prise en charge de GCC 4.7 [Bug #5851]
  • Autres correctifs

Voyez les tickets et le fichier ChangeLog pour plus de détails.
