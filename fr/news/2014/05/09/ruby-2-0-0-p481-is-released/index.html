Sortie de Ruby 2.0.0-p481

Posté par usa le 2014-05-09
Traduit par Jean-Denis Vauguet

La version 2.0.0-p481 de Ruby vient de sortir !

Cette nouvelle version incorpore un certain nombre de correctifs, tels que :

  • la possibilité de faire des build avec Readline-6.3
  • un correctif sur OpenSSL (suite à une régression dans p451)
  • une version plus à jour de libyaml (cf. Heap Overflow in YAML URI Escape Parsing (CVE-2014-2525)).

Pour plus de détails, vous pouvez consulter les tickets ainsi que le ChangeLog.
