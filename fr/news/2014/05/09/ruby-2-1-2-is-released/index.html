Sortie de Ruby 2.1.2

Posté par nagachika le 2014-05-09
Traduit par Jean-Denis Vauguet

La version 2.1.2 de Ruby vient de sortir !

Cette nouvelle version contient un certain nombre de correctifs :

  • régression de Hash#reject
  • support des build avec Readline-6.3 (cf. Bug #9578)
  • nouvelle version de libyaml with psych
  • de nombreux autres fix.

Pour plus de détails, vous pouvez consulter les tickets et le ChangeLog.
