Ruby 2.1.4 est disponible

Posté par nagachika le 2014-10-27
Traduit par Geoffrey Roguelon

Ruby 2.1.4 est disponible.

Cette version inclut des correctifs de sécurité pour les failles suivantes :

  • CVE-2014-8080 : Déni de service sur l'expansion XML
  • Changement des options par défaut de ext/openssl

De plus, plusieurs bugs ont été corrigés.

Voir les tickets et le ChangeLog pour plus de détails.

Mise à jour : Une régression introduite dans Ruby 2.1.3 est désormais corrigée :
