Ruby 2.0.0-p594 est disponible

Posté par usa le 2014-10-27
Traduit par Geoffrey Roguelon

Nous avons le plaisir de vous annoncer la sortie de Ruby 2.0.0-p594.

Cette version inclut un correctif de sécurité sur une faille DoS de REXML.

  • CVE-2014-8080 : Déni de service sur l'expansion XML

Cette version inclut également un changement sur les options par défaut de ext/openssl. Les options non-sécurisées de SSL/TLS sont désormais désactivées par défaut.

  • Changement des options par défaut de ext/openssl

De plus, plusieurs bugs ont été corrigés. Voir les tickets et le ChangeLog pour plus de détails.
