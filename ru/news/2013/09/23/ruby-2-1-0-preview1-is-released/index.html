Что нового

Значительные изменения в:

  • VM (кеш методов)
  • RGenGC (См. презентацию ko1 на RubyKaigi и на EuRuKo)
  • refinements
  • синтаксис
  • десятичные литералы (Decimal Literal)
  • постоянные строковые литералы (Frozen String Literal)
  • определение метода теперь возвращает значение
  • Bignum
  • 128bit
  • GMP
  • String#scrub
  • Socket.getifaddrs
  • новый RubyGems

See more details of changes Больше подробностей в новостях в репозитарии Ruby.
