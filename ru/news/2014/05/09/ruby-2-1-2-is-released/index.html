Вышел Ruby 2.1.2

Опубликовал nagachika 09-05-2014
Перевел: gazay

Мы рады сообщить о новом релизе: Ruby 2.1.2.

Этот релиз содержит фикс для регрессии в методе Hash#reject версии Ruby 2.1.1, поддержку сборки с Readline-6.3, (см. Bug #9578), обновление встроенных версий libyaml и psych, и несколько багфиксов.

Смотри тикеты и лог изменений для дальнейшей информации.
