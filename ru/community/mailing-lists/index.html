Почтовая рассылка

Почтовая рассылка – отличный способ держать руку на пульсе Ruby сообщества. {: .summary}

Существует четыре основных англоговорящих почтовых рассылки:

Ruby-Talk : Это наиболее популярная почтовая рассылка, рассматривая основные вопросы о Ruby. (Архив)

Ruby-Core : Эта рассылка для обсуждения ядра и внутреннего устройства Ruby. Часто используется для демонстрации патчей. (Архив)

Ruby-Doc : Данная рассылка служит для обсуждения инструментов и стандартов документации в Ruby. (Архив)

Ruby-CVS : Данная рассылка транслирует все коммиты в Subversion репозиторий Ruby.

Новостная группа comp.lang.ruby : Те, кто предпочитает Usenet почтовой рассылке, возможно захотят взглянуть на comp.lang.ruby новостную группу.
