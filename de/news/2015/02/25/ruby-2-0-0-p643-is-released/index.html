Ruby 2.0.0-p643 veröffentlicht

Geschrieben von usa am 25.2.2015
Übersetzt von Marvin Gülker

Wir freuen uns, die Veröffentlichung von Ruby 2.0.0-p643 bekannt geben zu können.

Hierbei handelt es sich um die letzte ordentliche Veröffentlichung von Ruby 2.0.0, welches nunmehr in die Phase der Sicherheitsunterstützung übergeht. Es werden daher keine neuen Versionen von Ruby 2.0.0 mehr veröffentlicht, sofern nicht kritische Regressions oder Sicherheitslücken gefunden werden. Die Dauer dieser letzten Phase ist auf 1 Jahr angesetzt. Danach wird die Unterstützung von Ruby 2.0.0 am 25. Februar 2016 enden. Wir empfehlen daher, mit den Planungen für eine Migration auf neuere Ruby-Versionen wie 2.1 oder 2.2 zu beginnen.

Diese Veröffentlichung enthält viele Fehlerkorrekturen. Siehe die Tickets und das ChangeLog für weitere Informationen.
