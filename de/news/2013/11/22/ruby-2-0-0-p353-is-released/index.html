Ruby 2.0.0-p353 veröffentlicht

Geschrieben von nagachika am 22.11.2013
Übersetzt von Quintus

Ruby 2.0.0-p353 ist veröffentlicht worden.

Dieses Release enthält einen Sicherheitsfix betreffs Gleitkommazahl-Parsing.

  • Heap-Überlauf beim Parsen von Floating-Point (CVE-2013-4164)

Außerdem wurden einige Fehler behoben. Siehe die Tickets und das ChangeLog für Details.
